## 🚀 Quick start

1.  **Set up the project.**

    ```shell
    # install the project dependencies
    yarn install
    ```

2.  **Start developing.**

    Navigate into your new site’s directory and start it up.

    ```shell
    cd alexmachina/
    yarn start
    ```

3.  **Open the code and start customizing!**

    Your site is now running at http://localhost:8000!
